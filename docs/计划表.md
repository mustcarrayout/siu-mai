# 周计划

## 什么是周计划?

1. 每周五制定计划;
2. 每周五进行复盘;
3. 迭代进行;
4. 根据历史对复盘计划下周或者下下周或者下个月要完成的事情;

关键点:

- 计划合理、小步快步、重复再重复
- 结束复盘
- 科学评估

## 周计划做什么?

- 根据问题进行计划安排
    - 按照紧急和重要程度进行安排
    - 按照上期结束复盘进行计划调整
    - 计划要完成且要认真完成
- 培养习惯
    - 做计划对习惯;
    - 如何合理地对某事物进行评估;
    - 多套方案构思(全面、系统、原问题出发);

## 如何合理对安排计划?

- 你的梦想是什么?
- 你今年想做什么?
- 你想过怎样的生活?
- 你对你的工作还满意吗?
- 你还需要提升什么?
- 如何评估自己的收益?
